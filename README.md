# smart_phone_case_NEQ
NEQ is designed for teenagers to help them prevent from neck pain due to overuse of smartphone.
NEQ includes a phone case and an app connected with it via bluetooth.
Phone case is built with arduino nano board, battery,led strip,servo motor and bluetooth module.
Led reflects different status of neck and movement of servo promotes users to adjust to a good neck posture.
App is designed based on Android system and already tested in Xiaomi 2s successfully.
The main functionality of app is to achieve 3-axis accelerometer data and send it to phone case via bluetooth.
Additionally app can help set goals and record progress.
